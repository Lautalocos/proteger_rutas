# Food Store — Evaluación 1 Programación III

Aplicación frontend de una tienda de comida. Desarrollada con Vite y TypeScript

link a presentacion: https://youtu.be/ILghnyXYmRc

## Alumno

Laner, Lautaro — Tecnicatura Universitaria en Programación — UTN

## Funcionalidades

- Catálogo de productos renderizado dinámicamente
- Búsqueda de productos por nombre en tiempo real
- Filtrado por categoría
- Carrito de compras con persistencia en localStorage
- Control de stock al agregar productos
- Sistema de autenticación con roles (admin y client)

## Cómo ejecutar

```bash
pnpm install
pnpm dev
```

Navegar a:
- Catálogo: http://localhost:5173/src/pages/store/home/home.html
- Carrito: http://localhost:5173/src/pages/store/cart/cart.html
